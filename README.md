# Pejic - Language System for Learning

**A comprehensive language learning and development system built with multiple support providers and instant feedback mechanisms.**

---

## 📋 Overview

Pejic is a complete system for learning and development with support for:
- **Pejic Language Support** - Language server with syntax highlighting and IntelliSense
- **Provider System** - 7 different providers for various functionalities
- **Instant Reaction System** - Real-time feedback on every code edit
- **VS Code Integration** - Direct integration with VS Code extensions
- **Security & Validation** - Security checks and code validation

---

## 📚 Language Levels (AIEOM)

The AIEOM system defines knowledge levels:

- **AIEOM: Full name** - Absolute Infinite Eternal Omega Master
- **AIEOM: Absolute** - absolute, unlimited, unchanging, and beyond all limitations, rules, and boundaries.
- **AIEOM: Infinite** - infinite, without any limitation in size, power, capabilities, or development.
- **AIEOM: Eternal** - eternal, without beginning or end, beyond time and transience.
- **AIEOM: Omega** - the last, final, and highest level; the endpoint of any hierarchy.
- **AIEOM: Master** - complete mastery; perfection in knowledge, skills, and execution.

- The AIEOM standard defines the highest design and development philosophy for the Pejic Ecosystem.

## 🛠️ Components

### 1. **Pejic Language** (`.pej`)
- Dark theme with 20 foreground colors
- Syntax highlighting
- Keyword support
- Forbidden words list

### 2. **Nexus Language Support** (`.nxs`)
- Syntax highlighting
- Code completion
- Error detection
- Language features

### 3. **Nexus Config** (`.nxc`)
- System configuration
- Compiler settings
- Optimization options

### 4. **Instant Reaction System** (`.nxr`)
- Instant feedback
- Error highlighting
- Success notifications
- Warning messages

### 5. **Provider System** (`.nxp`)
7 providers:
- **Syntax Provider** - Syntax validation
- **Language Provider** - Language services
- **Compiler Provider** - Code compilation
- **Config Provider** - Configuration management
- **Reaction Provider** - Feedback system
- **Theme Provider** - Theme settings
- **Security Provider** - Security checks

### 6. **VS Code Extensions**
- Pejic Language Support Extension
- Pejic Provider System Extension
- Pejic Complete Development Kit

## 🚀 Installation

### Method 1: Direct from VS Code
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Pejic"
4. Click "Install"

### Method 2: Via VS Code Extension URLs
```javascript
// Directly from vscodeExtensionUrls.js
- vscode:extension/Pejicc.pejic-language-support
- vscode:extension/Pejicc.pejic-provider-system
- vscode:extension/Pejicc.pejic-dev-kit
```

### Method 3: From VS Code Marketplace
Visit: [VS Code Marketplace](https://marketplace.visualstudio.com/)

## ⚙️ Configuration

### System Settings

```javascript
{
  version: '1.0.0',
  language: 'Pejic',
  fileExtensions: ['.pej', '.nxs', '.nxc', '.nxr', '.nxp'],
  theme: 'dark',
  coding: 'UTF-8'
}
```

### Compiler Options

```javascript
{
  strict: true,
  warnings: true,
  optimizations: true,
  sourceMap: true,
  minify: false
}
```

### Security

```javascript
{
  enabled: true,
  checkForbiddenWords: true,
  validateTypes: true,
  scanVulnerabilities: true
}
```
